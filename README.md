# Julian D Costa - Portfolio

A modern, clean, and professional portfolio website for **Julian D Costa**, a specialized Graphics Designer in Branding and Print Design.

## Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Modern UI**: Built with a clean blue and white corporate aesthetic using Tailwind CSS.
- **Interactive Portfolio**: Categorized gallery with a lightbox effect for viewing projects.
- **Dynamic Timeline**: Professional experience section showcasing career milestones.
- **Animated Components**: Smooth scroll effects and subtle animations for a premium feel.
- **Ready for Deployment**: Configured for easy hosting on platforms like GitHub Pages.

## Tech Stack

- **React**: For building a dynamic and component-based user interface.
- **Tailwind CSS**: For utility-first styling and rapid UI development.
- **Lucide React**: For beautiful, consistent iconography.
- **ES6 Modules**: Modern JavaScript structure for performance and maintainability.

## Sections

1. **Hero**: Quick introduction with professional CTA buttons.
2. **About**: Detailed bio and professional background.
3. **Skills**: Visual representation of expertise in design tools.
4. **Services**: Clear service offerings for branding and print.
5. **Portfolio**: A showcase of selected works with filtering options.
6. **Experience**: A timeline of professional roles and achievements.
7. **Contact**: Direct communication channels and a functional inquiry form.
